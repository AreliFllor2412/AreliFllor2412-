- Frontend: React (UI dinámica)
- Backend: FastAPI (procesamiento + IA)

- 📊 Generación automática de reportes  
- 🔔 Notificaciones automatizadas  
- 🔄 Integración completa frontend + backend

- Wav2Vec2 + Librosa + Python  

### 🎯 Enfoque
